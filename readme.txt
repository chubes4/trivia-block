Create interactive trivia questions with real-time scoring and customizable result messages using Gutenberg blocks.

== Description ==

**Trivia Block** transforms your WordPress site into an interactive learning experience! Create engaging trivia questions with multiple choice answers, automatic scoring, and custom result messages that adapt to your brand or theme.

Perfect for:
* Educational content and online courses
* Entertainment and festival websites (imagine "Bonnaroo Veteran" for music trivia!)
* Corporate training and knowledge checks
* Blog engagement and user interaction
* Community building through fun quizzes

= Key Features =

* **Easy Gutenberg Integration** - Native WordPress block editor support
* **Real-time Scoring** - Tracks correct answers across all questions on a page
* **Custom Result Messages** - Personalize the experience with themed result titles
* **Configurable Score Ranges** - Set your own thresholds for different performance levels
* **Mobile Responsive** - Works perfectly on all devices
* **Accessibility First** - Full keyboard navigation and screen reader support
* **Single Selection** - Prevents multiple attempts per question
* **Immediate Feedback** - Shows correct answers when users are wrong
* **Smooth Animations** - Professional, polished user experience

= How It Works =

1. Add a "Trivia Question" block in the Gutenberg editor
2. Enter your question with rich text formatting
3. Add 2-6 answer options
4. Mark the correct answer
5. Customize result messages and score ranges in the block settings
6. Publish and watch users engage with your content!

= Custom Result Messages =

Make your trivia unique with custom result messages:

* **Festival Quiz**: "Deadhead Scholar", "Dancing Bear", "New to the Lot"
* **History Quiz**: "Time Traveler", "History Buff", "Keep Learning"
* **Corporate Training**: "Expert", "Proficient", "Needs Review"

Result messages apply to ALL trivia blocks on a page and sync automatically across blocks.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/trivia-block/`, or install through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for "Trivia Question" in the Gutenberg block inserter
4. Start creating engaging trivia content!

== Frequently Asked Questions ==

= Can I have multiple trivia questions on one page? =

Yes! Add as many trivia blocks as you want. The scoring system tracks all questions on the page and shows a cumulative score.

= Can I customize the result messages? =

Absolutely! Each trivia block has settings to customize result messages and score ranges. Changes apply to all trivia blocks on the same page.

= Does it work on mobile devices? =

Yes, the plugin is fully responsive and works great on phones, tablets, and desktops.

= Is it accessible? =

Yes! The plugin follows accessibility best practices with proper ARIA labels, keyboard navigation, and screen reader support.

= Can users retake the trivia? =

Currently, users get one attempt per question. A reset functionality is planned for future releases.
